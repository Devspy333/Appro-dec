## About Appro dec

**Appro dec** is a powerful tool for decoding APK files. It allows you to extract and analyze the contents of Android applications, making it easy to inspect app structure, resources, and code.

### Key Features
- Decode and extract APK file contents
- Analyze app resources and assets
- View application structure
- Easy-to-use interface powered by AI Studio

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
